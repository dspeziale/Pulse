Interrogazione dati (P-04)

Query strutturata Pulse (filtri e aggregazioni) inoltrata alla Sonda via POST /probes/{id}/query. Non è una DSL raw OpenSearch (decisione API-01).

Parametri query
La query viene eseguita sui dati locali (OpenSearch) della Sonda selezionata. I sistemi disponibili si aggiornano automaticamente a destra al cambio di Sonda.
Inizio finestra temporale in formato ISO-8601 UTC.
Fine finestra temporale in formato ISO-8601 UTC.
Ogni filtro è {"field","op","value"}. Operatori: eq, neq, gt, gte, lt, lte, in, not_in, contains, matches.
Es. media per intervallo, uptime. Lasciare vuoto per la sola ricerca documentale.
Esempi pronti

Filtro per stato di errore lento:

[
  {"field":"status","op":"eq","value":"error"},
  {"field":"response_ms","op":"gt","value":500}
]

Aggregazioni media + uptime:

[
  {"type":"avg","field":"response_ms","interval":"1h"},
  {"type":"uptime"}
]
